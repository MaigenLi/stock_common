# show_kline.py 使用指南

> 股票 K 线查看工具，支持单个查询、批量处理、涨幅排序筛选

## 文件位置

```
~/.openclaw/workspace/stock_common/show_kline.py
```

## 运行方式

```bash
cd ~/.openclaw/workspace
python stock_common/show_kline.py --code sh600036 --days 5

# 或者从 stock_common 目录内直接运行
cd stock_common && python show_kline.py --code sh600036 --days 5
```

## 依赖

- `stock_common`（位于 `~/.openclaw/workspace/stock_common/`）
- 通达信离线日线数据（默认路径 `~/stock_data/vipdoc/`）
- 股票代码列表文件（默认 `/home/hfie/stock_code/results/stock_codes.txt`）

---

## 快速开始

### 基本用法

```bash
# 查看单个股票最近10天K线
python show_kline.py --code sh600036

# 指定日期的K线
python show_kline.py --code sh600036 --date 2026-04-02

# 查看最近5天
python show_kline.py --code sh600036 --days 5

# 带详细交易日信息（仅单码模式）
python show_kline.py --code sh600036 --date 2026-04-02 --detail
```

## 功能二：批量查询

### 从文件读取多个股票

```bash
python show_kline.py --file my_watchlist.txt --date 2026-04-02 --days 3
```

文件格式（每行一个代码，支持 `#` 注释）：

```
# 我的关注列表
sh600036
sh601318
sz000858
sz000001
```

> 如果不指定 `--file`，默认使用 `/home/hfie/stock_code/results/stock_codes.txt`（全市场股票）

## 功能三：涨幅筛选排序（核心功能）

在全市场股票中按涨幅排序，找到今日/指定交易日最强股票。

### 基础涨幅筛选

```bash
# 今日涨幅≥5%，取前50名
python show_kline.py --rank --date 2026-04-02 --gain 5.0 --top 50
```

### 将结果输出到文件

```bash
python show_kline.py --rank --date 2026-04-02 --gain 5.0 --top 50 --output top_codes.txt
```

然后可以用这个文件批量查看详情：

```bash
python show_kline.py --file top_codes.txt --days 5
```

### 涨幅 + 趋势放量双重筛选

在涨幅基础上叠加趋势放量条件（要求股票处于上升趋势 + 成交量放大）：

```bash
python show_kline.py --rank --date 2026-04-02 --gain 3.0 --top 20 --trend-volume
```

> 注意：趋势放量筛选会分两阶段执行，第一阶段只读最后2条记录快速筛选涨幅，第二阶段才对候选股票读完整历史分析趋势。约需 5 分钟。

### 涨幅 + 趋势放量 + 突破平台

在趋势放量基础上增加"今日收盘突破过去20天最高价"条件：

```bash
python show_kline.py --rank --date 2026-04-02 --gain 3.0 --top 20 --trend-volume --breakout
```

### 涨幅 + 换手率过滤

```bash
# 涨幅≥5% 且 换手率≥3%
python show_kline.py --rank --date 2026-04-02 --gain 5.0 --top 30 --turnover 3.0
```

## 完整参数参考

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--code`, `-c` | 单个股票代码 | - |
| `--file`, `-f` | 从文件批量读取（每行一个代码） | `stock_codes.txt`（全市场） |
| `--rank` | 启用涨幅筛选排序模式 | - |
| `--date`, `-d` | 指定日期 (YYYY-MM-DD) | 今天 |
| `--days` | 显示最近多少天 | 10 |
| `--detail` | 显示详细交易日信息（仅单码） | - |
| `--gain`, `-g` | 涨幅阈值（%） | 5.0 |
| `--top`, `-n` | 取前N名 | 50 |
| `--output`, `-o` | 将结果代码输出到文件 | - |
| `--workers`, `-w` | 并发数 | 20 |
| `--trend-volume` | 叠加趋势放量筛选 | - |
| `--breakout` | 要求突破20日平台 | - |
| `--turnover` | 换手率阈值（%） | 0（不限制） |

## 使用场景示例

### 场景1：盘后复盘 — 找今日涨停股

```bash
python show_kline.py --rank --date 2026-04-02 --gain 9.5 --top 100 --output zhangting.txt
```

### 场景2：盘后复盘 — 找涨幅5%以上且趋势放量的强势股

```bash
python show_kline.py --rank --date 2026-04-02 --gain 5.0 --top 30 --trend-volume --output strong.txt
```

### 场景3：盘中 — 关注已突破平台的强势股

```bash
python show_kline.py --rank --date 2026-04-03 --gain 3.0 --top 50 --trend-volume --breakout
```

### 场景4：查看自选股的近期走势

```bash
python show_kline.py --file my_stocks.txt --days 20
```

### 场景5：查看某只股票的今日详细数据

```bash
python show_kline.py --code sh600036 --date 2026-04-02 --detail
```

## 性能说明

| 模式 | 耗时 | 说明 |
|------|------|------|
| 单码查询 | <1s | 读本地文件 |
| 批量查询（全市场） | 约30s | 顺序读每个文件 |
| 涨幅筛选（纯涨幅） | ~30s | Phase 1 仅读最后2条 |
| 涨幅+趋势放量 | ~300s | Phase 2 读完整历史 |

---

## 注意事项

1. **日期格式**必须是 `YYYY-MM-DD`
2. **涨幅筛选**默认使用全市场 5192 只股票，--date 必须是交易日
3. 成交额字段使用 IEEE 754 浮点数解析，数据真实可靠
4. 批量模式下 `--detail` 仅在单个股票时生效
